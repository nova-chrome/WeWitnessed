# shadcn/ui

When working with shadcn components, fetch the documentation index for reference:

**Documentation Index**: https://ui.shadcn.com/llms.txt

## When to Fetch

- Adding a new component
- Unsure about component props or variants
- Looking for usage examples
- Implementing dark mode or theming
- Setting up forms with shadcn

## Quick Reference

- CLI: `npx shadcn@latest add <component>`
- Config: `components.json`
- Components location: `components/ui/`
- Utils: `lib/utils.ts` (cn helper)

## Installed Components

Check `components/ui/` for already installed components before fetching docs.
