## Vision

Build a wedding guest memory platform allowing guests to capture, upload, and preserve wedding memories while automatically generating emotional storytelling experiences for couples.

---

## Core Value Proposition

- Capture candid guest moments
- Automatically organize event memories
- Enable AI-powered memory storytelling
- Provide real-time guest engagement
- Maintain strong privacy and moderation controls

---

## Primary Differentiators

- Timeline-controlled capture
- AI memory story builder
- Low-friction guest onboarding
- Offline upload reliability
